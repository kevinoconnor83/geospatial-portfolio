# Geospatial Portfolio

A collection of real-world geospatial analysis, modeling, and data-driven projects focused on search and rescue, spatial decision support, and recruiting analytics.

## Featured Projects

### [LandSAR](landsar/README.md)
Terrain-aware search and rescue modeling designed to support real-world decision-making through geospatial analysis and probabilistic simulation.

Key areas:
- Monte Carlo movement simulation
- Terrain and slope-based modeling
- Route, trail, and access integration
- Search prioritization and movement corridor analysis

### [Recruiting Analytics](recruiting-analytics/README.md)
Spatial and trend-based analysis of Coast Guard recruiting activity in South Texas, focused on saturation patterns, geographic shifts, and four-year trend evaluation.

Key areas:
- Recruit distribution mapping
- Saturation and density analysis
- Year-over-year trend comparisons
- Regional decision support

## Project Structure

- `landsar/` — Search and rescue modeling, documentation, and visuals
- `recruiting-analytics/` — Recruiting trend analysis, maps, charts, and supporting documentation
- `assets/` — Shared visuals and supporting files
